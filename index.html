What the firmware does

Keep the nametag. Tap KEY2 for the counter.

01

Name badge and counter

Upload names with the BadgeMagic app. A short KEY2 press rotates through those names and a BT 00–BT 64+ counter. The long KEY2 ISP gesture is inherited from FOSSASIA, but still needs a physical test on this BIN.

02

Three-second passive scan

Scanning runs in either display mode. When the app is disconnected, FrogAlert pauses advertising for three seconds and listens for BLE advertisements. It does not pair, connect, request location, or write a cloud log.

03

Temporary alerts

A matching public OUI or advertised name shows COP DETECTED or FLIPPER DETECTED for three seconds. A BadgeMagic name or FEE0 service shows three dancing frogs. The nametag or counter returns afterward.

These matches are hints, not identification. BLE addresses rotate, names can be copied, vendors share components, and service UUIDs can be reused. Passive scans also miss names sent only in scan responses. The FEE0 fallback can match other compatible badges.

Timing

A scan starts about every 20 seconds.

The current private build scans for three seconds, records only an approximate count and matching alert class, then returns to the selected display. The build passes software audits but has not been tested on the badge.

It is not hosted or released.
